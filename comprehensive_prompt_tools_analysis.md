#### 기술 스택
- **프론트엔드**: Next.js 14, React 19, TypeScript
- **스타일링**: Tailwind CSS
- **배포**: Docker, Vercel, Cloudflare Pages
- **AI 통합**: OpenAI API, Anthropic API (커스터마이징 가능)

#### 주요 기능

**1. Text-to-Prompt (텍스트 → 프롬프트)**
- 사용자가 간단한 아이디어나 키워드를 입력하면 AI가 최적화된 미드저니 프롬프트로 확장
- 자연어 처리를 통해 모호한 설명을 구체적인 프롬프트로 변환
- 예시: "고양이" → "/imagine prompt: A majestic orange tabby cat with piercing green eyes, sitting on a windowsill bathed in golden hour sunlight, photorealistic style, serene mood --ar 16:9 --v 6"

**2. Image-to-Prompt (이미지 → 프롬프트)**
- 이미지를 업로드하면 AI가 이미지를 분석하여 상세한 텍스트 설명 생성
- 비전 AI 모델을 사용하여 색상, 구성, 스타일, 분위기 등을 파악
- 생성된 설명을 기반으로 미드저니 프롬프트 자동 생성
- 드래그 앤 드롭 및 붙여넣기 지원

**3. 프리셋 파라미터 선택**
- **종횡비**: 1:1, 2:3, 3:2, 9:16, 16:9, 21:9 등
- **아트 스타일**: Photorealistic, Oil Painting, Watercolor, Digital Art, 3D Render, Anime, Comic Book, Sketch, Abstract, Minimalist, Cyberpunk, Steampunk 등
- **무드**: Dramatic, Peaceful, Energetic, Mysterious, Joyful, Dark, Bright, Melancholic, Epic, Serene 등
- **버전**: V6, V5.2, V5.1, Niji 6 등
- **기타 파라미터**: --stylize, --chaos, --quality 등

**4. 실시간 미리보기**
- 프롬프트 생성 과정을 실시간으로 확인
- 파라미터 변경 시 즉시 프롬프트 업데이트
- 스트리밍 방식으로 AI 응답 표시

**5. 원클릭 복사**
- 생성된 프롬프트를 클립보드에 즉시 복사
- 복사 성공 시 시각적 피드백 제공
- 미드저니 Discord 또는 웹사이트에 바로 붙여넣기 가능

**6. 프롬프트 히스토리**
- 생성된 프롬프트 자동 저장 (로컬 스토리지)
- 이전 프롬프트 검색 및 재사용
- 즐겨찾기 기능

#### 설치 및 사용

**웹 데모 사용 (가장 쉬움)**
```
1. https://mj.u14.app/prompt 방문
2. 아이디어 입력 또는 이미지 업로드
3. 파라미터 선택
4. "Generate" 클릭
5. 생성된 프롬프트 복사
```

**로컬 실행**
```bash
# 저장소 클론
git clone https://github.com/Amery2010/midjourney-prompt-generator.git
cd midjourney-prompt-generator

# 의존성 설치
npm install

# 환경 변수 설정
cp .env.example .env
# .env 파일에 AI API 키 추가

# 개발 서버 실행
npm run dev

# 브라우저에서 http://localhost:3000 접속
```

**Docker 배포**
```bash
# Docker 이미지 빌드
docker build -t midjourney-prompt-generator .

# 컨테이너 실행
docker run -p 3000:3000 \
  -e OPENAI_API_KEY=your_key \
  midjourney-prompt-generator
```

**Vercel 배포 (원클릭)**
```
1. GitHub에 저장소 포크
2. Vercel 계정 생성
3. "Import Project" 클릭
4. GitHub 저장소 선택
5. 환경 변수 추가 (OPENAI_API_KEY 등)
6. "Deploy" 클릭
```

### 8. ChatGPT-Midjourney-Prompt-Generator ⭐⭐⭐⭐

#### 개요

ChatGPT를 미드저니 프롬프트 생성 전문가로 변환하는 메타 프롬프트입니다. 별도의 설치나 도구 없이 ChatGPT에 프롬프트를 입력하면 즉시 사용 가능합니다.

#### 주요 기능

**1. 메타 프롬프트**
- ChatGPT를 프롬프트 생성 전문가로 설정
- 미리 정의된 입력 및 출력 형식
- 일관된 품질의 프롬프트 생성

**2. 다중 변형 생성**
- 한 번의 입력으로 3개의 다른 프롬프트 생성
- 각 프롬프트는 다른 접근 방식 사용
- 사용자가 최적의 프롬프트 선택 가능

**3. 파라미터 자동 포함**
- 종횡비 (--ar)
- 버전 (--v)
- 스타일라이즈 (--stylize)
- 기타 미드저니 파라미터

**4. 컨텍스트 이해**
- 사용자의 의도 파악
- 모호한 설명을 구체적으로 변환
- 적절한 스타일 및 무드 제안

#### 사용 방법

**1. ChatGPT에 메타 프롬프트 입력**
```
Forget all previous prompts. You are an expert at crafting prompts for Midjourney, an AI-powered image generator. Your prompts result in stunning and original images. You understand composition, lighting, mood, and artistic styles.

When I give you a concept, you will create 3 different Midjourney prompts based on that concept. Each prompt should be detailed, creative, and optimized for Midjourney.

Format:
/imagine prompt: [detailed description], [style], [mood], [lighting], [composition], [additional details] --ar [aspect ratio] --v [version]

Now, create prompts for: [YOUR CONCEPT]
```

**2. 개념 입력**
```
A futuristic city at night
```

**3. ChatGPT 응답 예시**
```
1. /imagine prompt: A sprawling futuristic metropolis at night, towering skyscrapers with holographic advertisements, flying vehicles weaving between buildings, neon lights reflecting off rain-slicked streets, cyberpunk style, energetic mood, dramatic lighting with vibrant blues and purples, wide-angle composition, ultra detailed, 8k resolution --ar 16:9 --v 6

2. /imagine prompt: Futuristic cityscape under a starry night sky, sleek glass buildings with bioluminescent accents, elevated walkways connecting structures, soft ambient lighting from street lamps, sci-fi style, serene mood, cool color palette with teal and silver tones, bird's-eye view composition, highly detailed architecture --ar 21:9 --v 6

3. /imagine prompt: Night scene of a futuristic city, massive arcologies reaching into the clouds, light trails from high-speed trains, holographic trees lining boulevards, solarpunk style, hopeful mood, warm golden lighting mixed with cool blues, street-level perspective, photorealistic quality --ar 16:9 --v 6
```
